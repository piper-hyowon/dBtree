### 🌳 dBtree: 🍋 레몬나무에서 무료 데이터베이스를 수확하세요! 
: 가볍고 사랑스러운 무료 DBaaS

개발 기간: 2025-02 ~ (현재)

[기술 스택]
**Go** / **Kubernetes** / **React**


[주 사용자]
- 간단한 DB 환경을 선호하는 개발자
- 귀여운 디자인과 UX를 가진 DB 서비스를 찾는 개발자
- 무료 DB를 찾는 개발자


[주요 기능]
- 데이터베이스 서비스
  - Redis/MongoDB (기본/레플리카/클러스터 모드)
  - 간편한 설정으로 빠르게 시작
- 🍋
  - DB 퀴즈를 풀고 레몬을 수확해 크레딧으로 전환
  - 일정 시간마다 새로운 레몬이 자라남
  - 수확한 레몬으로 DB 인스턴스 생성 및 유지
